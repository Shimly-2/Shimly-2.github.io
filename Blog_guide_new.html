Overview

整理了从4000-8000￥的部分价位的部分笔记本电脑的性能配置，作为选购信息参考，大部分测试截图选自b站。

Requirements

  • 性价比高
  • 续航时间长
  • 重量轻
  • 性能不用太好（非游戏本）

How to choose

  • 看CPU：数字越大越好，字母U < HS < H, eg. R7-5800H < R7-6800U(即6800最拉跨的一版也要比5800最好的好)
  • 看续航：时间越长越好，AMD系列普遍对于续航更优化
  • 看重量：电脑本机+适配器重量越小越好
  • 看屏幕：屏幕分辨率越高越好，1080P的屏幕就别考虑了，太拉了
  • 看外扩：接口种类丰富好

再写一遍

  • 看CPU：数字越大越好，字母U < HS < H, eg. R7-5800H < R7-6800U(即6800最拉跨的一版也要比5800最好的好)
  • 看续航：时间越长越好，AMD系列普遍对于续航更优化
  • 看重量：电脑本机+适配器重量越小越好
  • 看屏幕：屏幕分辨率越高越好，1080P的屏幕就别考虑了，太拉了
  • 看外扩：接口种类丰富好

推荐：性能略差点，但是其他都不错

  • 追剧+轻度办公：华硕灵耀X13 锐龙版（接口太少，全是C口，续航好，屏幕质量高）

推荐：性能稍好，但是也有缺点

  • 办公+轻度渲染：荣耀 MagicBook14 2022锐龙版（续航好，接口少点，屏幕差点）
  • 办公+渲染+游戏：联想 ThinkBook 14+ 酷睿版（续航差点，接口多，可选独显，适配器重）

推荐：性能过剩

  • 办公+渲染+游戏：联想 YOGA PRO 14s（很贵，接口少点，屏幕质量高，屏幕可触摸，机身较重，cpu性能拉满，本身硬件配置拉满）
