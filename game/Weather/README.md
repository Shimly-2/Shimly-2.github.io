# Weather.io

## 一个使用 HTML、CSS、JS 和 API 制作的天气 Web 应用程序 🌞🌦

### 一个很棒的天气应用程序，它可以通过与位置相关的背景图像、玻璃化效果和倾斜效果来告诉您任何位置的天气。
使用 OpenWeatherMap API 制作天气，使用 Unsplash API 制作背景图片。
